# References

## Purpose
This folder collects supporting reference material for architecture and production decisions.

## What Is Here
- `ui-core.md` for UI-related guidance.
- `production/` for operational topics such as performance, security, and monitoring.

## When To Use
Use this folder when implementation or review requires context beyond core workflow rules.

## Scope Boundary
Do not place binding agent-routing rules or feature specifications in this folder.

## Maintenance
Keep references current, remove stale guidance, and avoid overlap with rule files in `rules/`.
