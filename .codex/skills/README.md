# Skills

## Purpose
This folder contains operational workflow artifacts used in day-to-day agent execution.

## What Is Here
- `prompts.md` as the prompt quick reference.
- Subfolders for `start`, `product`, `playbooks`, `qa`, and `references`.

## When To Use
Use this folder during kickoff, scoping, implementation prep, QA handoff, and reference lookups.

## Scope Boundary
Do not place routing definitions or global policy rules in this folder.

## Maintenance
Keep content practical and current; when process changes occur, verify kickoff and handoff consistency first.
